## 如何贡献技能
----
> * 在这个文件夹下，有一个文件，叫jineng.py
> * 打开它
> * 点击编辑按钮（打开文件后，文本区域右上角有编辑）
> * 先写一个class，class的名字就是技能名
> * 请在这个class里写，不要超出class，影响其他人贡献技能
> * 技能写完请在class的末尾注释上你的联系方式
> * 然后提交、
> * 通过是黑色，没通过是红色，审核中是绿色
> * 通过后我会进行适配，争取在下一个版本把已经审核通过的技能适配到机器人当中
